# Logistic Regression Assignment – Titanic Dataset

This repository contains a Logistic Regression model built on the Titanic dataset
as part of a Data Science practical assignment.

The aim of this assignment is to predict whether a passenger survived or not
based on given features.

## Work Done
- Loaded the Titanic training dataset
- Performed basic exploratory data analysis (EDA)
- Handled missing values
- Removed unnecessary text columns
- Encoded categorical variables
- Built a Logistic Regression model
- Evaluated the model using accuracy, precision, recall, F1-score, and ROC-AUC
- Used the test dataset to predict survival for unseen data

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab
